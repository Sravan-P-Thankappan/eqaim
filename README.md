## Tech Stack

**Frontend:** React.js,Tailwind CSS

**Backend:** Node, Express.js




## Run Locally

Clone the project

```bash
  https://github.com/Sravan-P-Thankappan/eqaim_task.git
```

Go to the project directory

```bash
  cd client (frontend)
  cd server (backend)
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
 client : npm start
 server : npm start

```
